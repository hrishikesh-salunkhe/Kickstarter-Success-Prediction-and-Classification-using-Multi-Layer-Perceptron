KickStarter Project Predicting Tool

About Project:

Kickstarter is one of the leading crowdfunding platforms that helps entrepreneurs get funding from the people.But data shows that every one out of three projects on Kickstarter fail to receive their funding goal till they reach their deadline.
Our Web Tool aims at minimizing this failure by classifying whether a project is likely to be successful or not by using our Multi-Layer Perceptron algorithm.
The tool also allows Project Creators to analyse how much funding they are likely to recieve on their project with help of Linear Regression and to top it off we have provided statistics by performing Exploratory Data Analysis on 600,000+ KickStarter Projects.

Datasets Used:

  1. KickStarter General Dataset:https://www.kaggle.com/kemical/kickstarter-projects
  2. KickStarter Content Dataset:https://www.kaggle.com/tayoaki/kickstarter-dataset
  3. KickStarter Description Dataset:https://www.kaggle.com/antonionoca/kickstarter2018nlp
  4. KickStarter Rewards Dataset:https://www.kaggle.com/parienza/kickstarter
